Closing Tags

Closing tags are used to properly end element that contain content

Example

p ,h1 to h6, ul, ol, li, table, tr, td, th

These are example of closing tags

Self closing tags

These are the html tags that don't have content inside,so they don't need a closing tag

Example

br/, hr/, img/, input/

these are example of self closing tags

Empty tags

An empty tag is an html element that cannot have any content inside it no text inside so no closing tag needed

Example

br, hr, img, input

These are example of empty tags
