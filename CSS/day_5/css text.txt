*class selector:

*id selector:

What is the purpose of universal selector in CSS?

The purpose of the universal selector in CSS is to select and apply styles to all elements within a web page

What is the difference between block and inline elements

block element:

starts with new line
occupy full width of the screen
can set width and height

inline element:

does not starts with new line
occupy the width of the text
can't set height and width


What is Flexbox and why is it used?

Flexbox (Flexible Box Layout) is a CSS layout module designed to arrange items in a single dimension (as a row or column)

What is the difference between justify-content and align-items?

-justify-content aligns items along the main axis (horizontal by default)
-align-items aligns items along the cross axis (vertical by default

What is the difference between px, em, rem, and percentage units?

px	Fixed size
em	Relative to parent font-size
rem	Relative to root (html) font-size
%	Relative to parent size

What is the difference between RGB, RGBA, and HEX color values in CSS?

RGB	Red, Green, Blue
RGBA	RGB + transparency
HEX	Hexadecimal color code
