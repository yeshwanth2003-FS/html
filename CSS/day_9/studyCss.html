SELECTORS

Element Selector

Selects all elements of a specific HTML tag

Example: h1, p, div

Class Selector

Selects elements with a specific class name

Example: .container

ID Selector

Selects an element with a unique ID.

Example: #header

Universal Selector

Selects all elements on the page.

Example: *

Group Selector

Applies the same style to multiple elements

Example: h1, h2, p

Box Model

The CSS Box Model is a fundamental concept used to design and control the layout of elements on a webpage. Every HTML element is treated as a rectangular box, and the Box Model defines how the size and spacing of that box are calculated

Padding

Padding is the space between the content and the border. It creates inner spacing inside the element and increases the overall size of the box

Border

The border surrounds the padding and content. It helps define the visible boundary of an element and can have different colors, styles, and thicknesses

Margin

Margin is the outer space outside the border. It creates distance between the element and other elements on the webpage

Flexbox

Flexbox (Flexible Box Layout) is a CSS layout system used to arrange and align elements efficiently within a container. It helps developers create responsive and flexible web layouts without using floats or complicated positioning techniques

Grid

CSS Grid is a powerful two-dimensional layout system used to create web page layouts using rows and columns. It allows developers to arrange elements in a structured and organized manner with precise control over spacing and positioning

Common Flex and Grid Properties

display: flex :

Turns a container into a flex container

flex-direction :

Controls the direction of items

justify-content :

Aligns items along the main axis.

align-items :

Aligns items along the cross axis.

flex-wrap :

Allows items to move to the next line when space is insufficient.

gap :

Creates spacing between flex items

display: grid :

Converts a container into a grid container.

grid-template-columns :

Defines the number and size of columns.

grid-template-rows :

Defines the number and size of rows.

gap :

Creates spacing between grid items

justify-content :

Aligns the entire grid horizontally.

align-items :

Aligns grid items vertically

grid-column :

Makes an item occupy multiple column

grid-row :

Makes an item occupy multiple rows

CSS Animation

CSS Animation is a technique used to add motion and visual effects to HTML elements without using JavaScript. It helps make web pages more interactive, attractive, and engaging for users. Animations can be used to move, rotate, scale, fade, or change the appearance of elements over time
