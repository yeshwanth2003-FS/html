1. What is DOM?

Answer:

DOM (Document Object Model) is a programming interface that represents an HTML document as a tree of objects. It allows JavaScript to access and manipulate HTML elements, CSS styles, and page content dynamically.

Uses of DOM

  • Change text
  • Change styles
  • Hide/Show elements
  • Handle button click events

2. What is React?

Answer:

React is a JavaScript library developed by Meta for building fast, interactive, and reusable user interfaces using components.

Features of React

  • Component-Based Architecture
  • Virtual DOM
  • Reusable Code
  • Fast Rendering
  • Single Page Applications (SPA)

3. What are Components?

Answer:

Components are reusable pieces of UI in React. Instead of writing the same HTML multiple times, we create a component once and reuse it wherever needed.

Examples

  • Navbar
  • Footer
  • Login Form
  • Product Card
  • Button

4. What are Functions?

Answer:

A function is a reusable block of code that performs a specific task. It executes only when it is called.

Advantages

  • Avoids repeating code
  • Makes code reusable
  • Improves readability
  • Easier to maintain
