Why Callback?

  1. Execute one function after another function completes.
  2. Used for asynchronous operations.
  3. Pass a function as an argument to another function

Why Promise?

  1. Avoid Callback Hell
  2. Handle asynchronous operations in a structured way
  3. Supports success (then) and failure (catch) handling

Why Async/Await?

  1. Makes Promise code cleaner and easier to read.
  2. Looks like synchronous code
  3. Simplifies error handling using try...catch

Why Fetch?

  1. Used to communicate with APIs.
  2. Retrieve (GET) and send (POST, PUT, DELETE) data.
  3. Returns a Promise.
